# demo-pgsimulator-pages

本仓库用于归档和发布多个独立 HTML 页面。这些页面通常用于演示、测试或作为静态资源被外部引用。

## 项目简介

这是一个纯前端页面仓库，不依赖后端服务。仓库内容以 HTML 文件为主，每个文件均可独立运行或通过 GitHub Pages 直接访问。适用于快速分享页面原型、功能演示片段或临时工具页面。

## 目录说明

```
/
├── index.html          # 仓库首页/导航页（可选）
├── page-xxx.html       # 示例页面
├── assets/             # 公共资源目录（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- 根目录下存放各独立 HTML 页面文件。
- `assets/` 目录存放页面共用的样式、脚本和图片资源。
- 每个 HTML 文件应自包含或明确引用 `assets/` 下的资源。

## 页面归档说明

- 本仓库仅用于存放静态 HTML 页面，不涉及任何后端逻辑或数据库。
- 页面内容以通用演示、功能测试或技术验证为目的，不针对特定网站或服务。
- 每次提交应保持页面可独立运行，避免破坏性变更。

## 维护说明

- 添加新页面时，请确保文件命名清晰且不与其他页面冲突。
- 修改公共资源时，注意检查是否影响已有页面的正常显示。
- 如需删除页面，建议先确认无外部引用后再操作。
- 欢迎通过 Issue 或 Pull Request 提出改进建议。

## 许可

本仓库内容仅供学习和演示使用。如需引用或分发，请遵守相关法律法规并注明出处。
